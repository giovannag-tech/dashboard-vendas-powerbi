# Dashboard de Vendas com Power BI

## Objetivo

Dashboard desenvolvido para análise de vendas utilizando Power BI.

## Indicadores

- Faturamento Total
- Quantidade Vendida
- Ticket Médio

## Análises Realizadas

- Faturamento por Região
- Quantidade Vendida por Produto
- Evolução das Vendas
- Filtro por Categoria

## Ferramentas Utilizadas

- Power BI
- Excel
- DAX

## Aprendizados

Neste projeto foram aplicados conceitos de modelagem de dados, criação de medidas DAX, indicadores de desempenho (KPIs) e visualização de dados para apoio à tomada de decisão.
